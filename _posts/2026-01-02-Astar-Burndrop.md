## 現在価格の算出方法

時価総額（Market Cap）と流通供給量（Circulating Supply）から現在価格を算出できます。

**計算式：**
```
価格 = 時価総額 ÷ 流通供給量
```

**実際の計算例：**

```
¥13,601,688,833.18(Market Cap) ÷ 8,262,288,140(Circulating Supply) = ¥1.646
```

## Burndropとは?

 * ユーザが保有しているAstr tokenを任意でBurnしてもらってのToken価値を高める試みです。
 * Astr tokenをBurnしたユーザには、Soneium関連のtokenが配布されます。

## どれだけのAstrがBurnされれば、Astr tokenの価値は向上するのか

時価総額が変わらないと仮定した場合、Burn量が増えると流通供給量が減り、価格が上昇します。

**計算式：**
```
新しい価格 = 時価総額 ÷ (流通供給量 - Burn量)
価格上昇率 = ((新しい価格 - 現在の価格) ÷ 現在の価格) × 100
```

## 結論

時価総額が変わらないと仮定した場合、**半分（50%）のAstrがBurnされると、価格は約2倍（100%上昇）になります**。

これは数学的に必然的な結果です：

```
時価総額 = 価格 × 流通供給量
```

時価総額が一定の場合、流通供給量が半分になれば、価格は2倍になる必要があります。

**具体例：**
- 現在: 8,262,288,140 ASTR × ¥1.646 = ¥13,601,688,833
- 50%Burn後: 4,131,144,070 ASTR × ¥3.292 = ¥13,601,688,833

つまり、**供給量が半分になると、価格は2倍になる**という関係が成り立ちます。

ただし、これは時価総額が変わらないと仮定した場合の理論値です。実際の市場では：
- Burnによる期待が時価総額を増やす可能性（買い需要の増加）
- 逆に、Burnが期待外れだった場合、時価総額が減る可能性

など、様々な要因が影響するため、実際の価格動向は理論値とは異なる可能性があります。
